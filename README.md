## 📦 Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/naveen-portfolio.git
```

2. Navigate to project folder:
```bash
cd naveen-portfolio
```

3. Install dependencies:
```bash
npm install
```

4. Run development server:
```bash
npm run dev
```

5. Build for production:
```bash
npm run build
```
